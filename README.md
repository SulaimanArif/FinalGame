## Shipwrecks and Mystical Islands

Welcome, survivors!
In this game, you must explore, build, find food, and survive for 7 days while your enemies get increasingly more deadly. Gear up and survive until help arrives!

### Description

This is an island survival game where you need to:

- Gather resources and collect food
- Build bases and craft tools
- Defend yourself from increasingly difficult enemies

After 7 days of survival, you win!

### Controls

WASD and mouse controls for movement
E – Open inventory
Hammer required – Build structures


Note: This is a standalone build for Windows, Mac, or Linux.
